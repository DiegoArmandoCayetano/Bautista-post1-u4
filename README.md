# 📌 Galería Interactiva — Unidad 4

## 📖 Descripción del Proyecto
Este proyecto consiste en una galería interactiva de tarjetas desarrollada con JavaScript puro (sin librerías externas). 

La aplicación permite:
- Crear tarjetas dinámicamente
- Eliminar tarjetas
- Filtrar tarjetas por categoría
- Visualizar un contador de tarjetas visibles
- Mostrar un mensaje cuando la galería está vacía

Se aplican conceptos de:
- Manipulación del DOM
- Eventos en JavaScript
- Delegación de eventos
- ES6 (arrow functions, template literals, desestructuración)
